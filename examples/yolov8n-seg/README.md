# YOLOv8n-seg demo

This demo follows the same command style as detection demos:

```text
image -> preprocess -> input bin -> Neuron RuntimeV2 -> 10 raw output bins -> decode + NMS + mask proto compose -> result image/json
```

## Models

- `models/yolov8n-seg/int8/`
  - `yolov8n-seg_int8.dla`
  - `yolov8n-seg_mtk_int8.tflite`
- `models/yolov8n-seg/fp32/`
  - `yolov8n-seg_fp32.dla`
  - `yolov8n-seg_mtk_fp32.tflite`

## Build

```bash
cmake -S . -B build
cmake --build build -j
```

## Run

Run INT8 model:

```bash
./build/yolov8n-seg_demo
```

Run FP32 model:

```bash
./build/yolov8n-seg_demo --fp32
```

Run with specific images:

```bash
./build/yolov8n-seg_demo \
  --image assets/images/bus.jpg \
  --image assets/images/zidane.jpg
```

Run without `NeuronRuntimeV2_run` (still loads the `.dla`, prints each input/output buffer size in bytes, letterbox-preprocesses images, and checks the buffer matches `input[0]`):

```bash
./build/yolov8n-seg_demo --skip-runtime
```

## Generated Artifacts

- `outputs/yolov8n-seg/detections/<image>.json`
- `outputs/yolov8n-seg/vis/<image>_seg.png`

## Implementation Notes

- Input is `1x3x640x640` NCHW.
- Ten outputs are grouped as:
  - mask coeff heads: `1x32x80x80`, `1x32x40x40`, `1x32x20x20`
  - cls logits heads: `1x80x80x80`, `1x80x40x40`, `1x80x20x20`
  - bbox DFL heads: `1x64x80x80`, `1x64x40x40`, `1x64x20x20`
  - proto: `1x32x160x160`
- Postprocess does DFL decode (`reg_max=16`), class sigmoid, confidence filter, class-wise NMS,
  then keeps matched 32-dim mask coefficients for each kept detection.
- Instance mask is generated by `coeff @ proto`, then sigmoid + bbox crop + inverse letterbox mapping
  to original image size.
- In INT8 mode, each output tensor is dequantized to float before DFL, sigmoid, NMS and mask compose.
